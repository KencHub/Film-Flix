# 🎬 FILM FLIX

> **Your go-to destination for discovering trending, popular, and all-time great movies — all in one place.**

Live Demo → [film-flix-gold.vercel.app](https://film-flix-gold.vercel.app)

---

## 📖 Overview

**Film Flix** is a sleek, responsive movie discovery web application that lets users explore movies across multiple curated categories. Whether you want to catch up on what's trending, browse the most popular titles, or explore a full movie catalog, Film Flix gives you a beautiful, fast interface to do it all.

Each movie card surfaces key details — release date, genre, and rating — with an option to download or view more information through an elegant modal popup.

---

## ✨ Features

- 🔍 **Search** — Quickly find any movie by title
- 🆕 **Recent Movies** — Stay up to date with the latest releases
- 🔥 **Popular Movies** — Discover what everyone is watching right now
- 🎞️ **All Movies** — Browse the full catalog with pagination (Previous / Next)
- 🪟 **Movie Detail Modal** — View release date, genre, and rating at a glance
- 📥 **Download** — Direct download action from the movie detail view
- 📱 **Responsive Design** — Works seamlessly across desktop and mobile

## 🚀 Getting Started

### Prerequisites

- Node.js (if running a dev server locally)
- A modern web browser

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/film-flix.git

# Navigate into the project directory
cd film-flix

# Open index.html directly in your browser, or serve it locally
open index.html
```

If the project uses a package manager:

```bash
npm install
npm run dev
```

---

## 🎯 How It Works

1. On load, the app fetches movie data from a movie database API.
2. Movies are sorted and displayed under three sections: **Recent**, **Popular**, and **All Movies**.
3. Users can **search** by title to filter results in real time.
4. Clicking a movie opens a **modal** with detailed info: poster, release date, genre, and rating.
5. Pagination controls let users navigate through the full catalog.

## 🌐 Deployment

This app is deployed on **Vercel** for fast, global delivery.

To deploy your own fork:

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or connect your GitHub repository directly on [vercel.com](https://vercel.com) for automatic deployments on every push.
